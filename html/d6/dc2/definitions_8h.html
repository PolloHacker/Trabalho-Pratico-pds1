Descrição detalhada

Este arquivo contém as definições das estruturas e variáveis globais usadas na simulação.

Este arquivo é um cabeçalho que define as estruturas usadas e permite a referenciação inter-arquivos das variáveis globais usadas. Contém as definições de:

  • estruturas:
    • para o jogador (TJogador)
    • para os pokemons (TPokemon)
  • variáveis globais usadas na gravação dos logs.
