Descrição detalhada

Tipo de variável estruturada para os jogadores.

Este tipo de variável define um jogador. Contém as seguintes informações:

  • num_poks: quantidade de pokemons do jogador
  • current_pok: indice do pokemon atual
  • poks: ponteiro para um array de pokemons
