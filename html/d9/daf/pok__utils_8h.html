◆ ataca()

void ataca ( TJogador * jogadores,
int atacante )

Realiza um ataque entre dois jogadores.

Para realizar o ataque, a função verifica qual a relação de tipo entre os pokemons envolvidos. Se o atacante for mais forte do que o defensor, o ataque é de 1.2x. Se o atacante for mais fraco do que o defensor, o ataque é de 0.8x. Ao final, verifica se o defensor foi derrotado e registra o resultado nos logs.

Parâmetros
jogadoresArray contendo os jogadores.
atacanteÍndice do jogador atacante.

◆ cria_jogadores()

TJogador * cria_jogadores ( char * data)

Cria uma lista de jogadores a partir de uma cadeia de caracteres.

Autor
Isaac Reyes

Esta função quebra uma cadeia de caracteres para criar um array de estruturas TJogador. A cadeia de caracteres deve conter informações sobre dois jogadores e seus Pokemons. Cada linha da cadeia de caracteres representa um Pokemon e seus atributos. A primeira linha da cadeia de caracteres deve conter o número de Pokemons de cada jogador.

Observação
Cada jogador é representado por uma estrutura TJogador.
Cada pokemon de cada jogador é representado por uma estrutura TPokemon.
Parâmetros
dataA string de dados contendo as informações dos jogadores e seus pokémons.
Retorna
Um ponteiro para a lista de jogadores criada.
