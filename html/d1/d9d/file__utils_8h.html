◆ cria_log()

void cria_log ( )

Cria um arquivo de log com o timestamp atual.

Cria um arquivo de log com o timestamp atual no formato log_d-m-Y_H-M-S.txt, onde d representa o dia, m representa o mês, Y representa o ano, H representa a hora, M representa o minuto e S representa o segundo.

Observação
O arquivo de log será criado no mesmo diretorio do executável.

◆ divide_linha()

char ** divide_linha ( char * line)

Divide uma cadeia de caracteres em um array de palavras.

Recebe uma cadeia de caracteres como entrada e a divide em palavras individuais. As palavras são armazenadas em um array alocado dinamicamente.

Observação
O último elemento do array é definido como NULL.
Parâmetros
lineA cadeia de caracteres de entrada a ser dividida.
Retorna
Um array de cadeia de caracteres alocado dinamicamente contendo as palavras.

◆ le_arquivo()

char * le_arquivo ( char * nome_arq)

lê um arquivo e retorna seu conteúdo como uma cadeia de caracteres

Lê dados de um arquivo especificado pelo nome e os retorna como uma cadeia de caracteres. Enquanto o número de pokemons lidos for menor que o número de pokemons especificado, os dados são lidos e concatenados em uma cadeia de caracteres.

Observação
Se o arquivo não puder ser aberto ou ocorrer um erro durante a leitura, NULL é retornado.
Parâmetros
nome_arqO nome do arquivo a ser lido.
Retorna
Uma cadeia de caracteres alocada dinamicamente contendo os dados lidos do arquivo.
